# Data directory

| Path | Contents |
|------|----------|
| `populated/` | Example patient JSON and generated OWL files (`ado_jane_doe_001.owl`, etc.) |
| `gold/` | Track 2 inter-annotator gold (`statements.json`, `annotator_a.json`, `annotator_b.json`) |
| `reference_ads/` | Reference advance directive PDFs (not redistributed in public repos) |

Paths are defined in `ado/paths.py`.
